Ecommerce Website

This was a project I started back in March 2020 when I first learned python. I was a part of a team when we made this project, I have worked with many features like payment integration using PayPal, login register, account activation using email.

The application displays different products which can be purchased. Users can rate the product and track their order and details in the Dashboard.

The payment details and receipt are displayed at the end of the payment and users can take a screenshot or else it will be sent to their registered email.

The Application is Fully ready to use as an Online Ecommerce Website with all the features which an Ecommerce Web App requires.
